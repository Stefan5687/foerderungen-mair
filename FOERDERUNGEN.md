# Förderprogramme Übersicht

## EU-Ebene

### EFRE (Europäischer Fonds für regionale Entwicklung)

| | |
|---|---|
| **Was** | Regionalförderung für Investitionen und Innovation |
| **Wer** | KMU in strukturschwachen Regionen (Südtirol) |
| **Förderung** | bis 50% der förderfähigen Kosten |
| **Maximal** | 200.000-500.000€ (je nach Programm) |
| **Typ** | Zuschuss |

---

### ESF+ (Europäischer Sozialfonds)

| | |
|---|---|
| **Was** | Arbeitsmarkt, Weiterbildung, soziale Integration |
| **Wer** | Unternehmen mit Weiterbildungsbedarf |
| **Förderung** | bis 100% (abhängig vom Programm) |
| **Typ** | Zuschuss |

---

### Horizon Europe

| | |
|---|---|
| **Was** | Forschungs- und Innovationsförderung |
| **Wer** | Unternehmen mit F&E-Projekten |
| **Förderung** | 70-100% der Projektkosten |
| **Typ** | Zuschuss (förderfähige Kosten) |

---

## National (Italien)

### Credito d'Imposta per Ricerca e Sviluppo (Forschungszulage)

| | |
|---|---|
| **Was** | Steuererleichterung für F&E-Ausgaben |
| **Wer** | Unternehmen mit Forschungsaktivitäten |
| **Förderung** | **bis 35%** der förderfähigen Kosten |
| **Förderfähig** | Personalkosten, Material, Patente, Beratung |
| **Maximal** | 20 Mio.€/Jahr |
| **Typ** | Steuerguthaben |

---

### Transizione 4.0 (Investitionsanreize)

| | |
|---|---|
| **Was** | Förderung für Digitalisierung und Innovation |
| **Wer** | Investierende Unternehmen |
| **Förderung** | **bis 50%** (je nach Investition) |
| **Förderfähig** | Maschinen, Software, Patente |
| **Typ** | Steuerguthaben |

---

### Innovazione PMI

| | |
|---|---|
| **Was** | Innovationsförderung für kleine Unternehmen |
| **Wer** | KMU |
| **Förderung** | bis 20% der Investitionskosten |
| **Typ** | Zuschuss |

---

### Credito d'Imposta Mezzogiorno

| | |
|---|---|
| **Was** | Investitionsförderung für den Süden Italiens |
| **Wer** | Unternehmen in Süditalien (NICHT Südtirol!) |
| **Achtung** | **Südtirol nicht förderfähig** |

---

## Landes-Ebene (Südtirol)

### Wirtschaftsförderung (Landesbeiträge)

| | |
|---|---|
| **Was** | Zuschüsse für Investitionen und Expansion |
| **Wer** | KMU in Südtirol |
| **Förderung** | 10-30% der Investitionskosten |
| **Typ** | Zuschuss |

---

### Tourismus-Förderung

| | |
|---|---|
| **Was** | Förderung für Hotels, Gastronomie, Tourismus |
| **Wer** | Tourismusbetriebe in Südtirol |
| **Förderung** | 20-50% (je nach Maßnahme) |
| **Förderfähig** | Renovierung, Ausstattung, Digitalisierung |
| **Typ** | Zuschuss |

---

### Landwirtschafts-Förderung

| | |
|---|---|
| **Was** | Förderung für landwirtschaftliche Betriebe |
| **Wer** | Bauern, Agrarbetriebe |
| **Förderung** | 30-70% (je nach Maßnahme) |
| **Förderfähig** | Maschinen, Gebäude, Umstellung |
| **Typ** | Zuschuss |

---

### Umwelt- und Energie-Förderung

| | |
|---|---|
| **Was** | Förderung für Umwelt- und Klimaschutz |
| **Wer** | Unternehmen mit Investitionsprojekten |
| **Förderung** | 20-50% |
| **Förderfähig** | Photovoltaik, Effizienz, erneuerbare Energien |
| **Typ** | Zuschuss |

---

### Digitalisierungs-Förderung

| | |
|---|---|
| **Was** | Förderung für IT und Digitalisierung |
| **Wer** | Unternehmen aller Branchen |
| **Förderung** | 20-50% |
| **Förderfähig** | Software, Hardware, Schulungen |
| **Typ** | Zuschuss |

| Landesförderungen | 10-50% | Variabel |
