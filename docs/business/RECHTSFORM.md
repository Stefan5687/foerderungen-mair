# Rechtsform: Ditta Individuale (Einzelunternehmer)

## Südtirol/Italien – Was du wissen musst

---

## 1. Rechtsform wählen

### Ditta Individuale (Empfohlen)

| | |
|---|---|
| **Definition** | Einzelunternehmer |
| **Stammkapital** | Keines erforderlich |
| **Haftung** | Unbeschränkt (privates Vermögen) |
| **Steuern** | Einkommensteuer (IRPEF) + IVA |

---

## 2. Anmeldung – Schritt für Schritt

### Schritt 1: Partita IVA beantragen

**Partita IVA** = Umsatzsteuernummer (italienisch)

**Wo beantragen:**
- Agenzia delle Entrate (Finanzamt)
- Online: https://telematici.agenziaentrate.gov.it/
- Oder persönlich: Bozen, Agentur für Einnahmen

**Benötigt:**
- Personalausweis/Pass
- Steuernummer (Codice Fiscale)
- Antragsformular (Modello AA9/12)

**Kosten:** Kostenlos

---

### Schritt 2: Handelsregister (Camera di Commercio)

**Wo:** Camera di Commercio di Bolzano (https://www.bs.camcom.gov.it/)

**Benötigt:**
- Partita IVA
- Personalausweis
- Anmeldung (Comunicazione Unica)

**Kosten:** ca. 100-200€/Jahr (Handelskammerbeitrag)

---

### Schritt 3: INPS (Sozialversicherung)

**INPS** = Istituto Nazionale della Previdenza Sociale

**Anmeldung:**
- Online: www.inps.it
- Oder persönlich

**Beitrag (geschätzt):**
- Ca. 25-35% des Gewinns
- Mindestbeitrag: ca. 3.000-4.000€/Jahr

---

### Schritt 4: Gewerbeanmeldung (Comune)

**Wo:** Gemeinde (Comune) deines Standorts

**Benötigt:**
- Personalausweis
- Partita IVA
- Eventuell: Mietvertrag oder Eigentumsnachweis

**Kosten:** ca. 50-150€/Jahr (Gewerbesteuer)

---

## 3. Übersicht: Kosten pro Jahr

| Posten | Geschätzte Kosten/Jahr |
|---|---|
| Handelskammer (Camera di Commercio) | 100-200€ |
| INPS (Sozialversicherung) | 3.000-4.000€ |
| Gewerbeanmeldung (Comune) | 50-150€ |
| Buchhaltung/Steuerberater | 500-1.500€ |
| **Gesamt (geschätzt)** | **~4.000-6.000€/Jahr** |

---

## 4. Steuern

### Einkommensteuer (IRPEF)

| Einkommen | Steuersatz |
|---|---|
| bis 8.174€ | 23% |
| 8.174€ - 28.174€ | 25% |
| 28.174€ - 45.609€ | 35% |
| über 45.609€ | 43% |

### IVA (Mehrwertsteuer)

| | |
|---|---|
| **Normalsatz** | 22% |
| **Reduzierter Satz** | 4%, 5%, 10% |

**Für Beratungsleistungen:** 22% IVA
