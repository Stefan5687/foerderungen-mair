## Vertriebskanäle

### 1. E-Mail-Kaltakquise

| | |
|---|---|
| **Methode** | E-Mails an potenzielle Kunden |
| **Tools** | Gmail, Klaviyo, Mailchimp |
| **Volumen** | 100-500 E-Mails/Woche |
| **Erfolgsrate** | 2-5% Antworten |

**Prozess:**
1. Liste erstellen ( Website-Scraping, Wirtschaftskammer)
2. E-Mail senden
3. Follow-up nach 5 Tagen
4. Termin vereinbaren

---

### 2. LinkedIn

| | |
|---|---|
| **Methode** | Direktnachrichten, Posts |
| **Tools** | LinkedIn, LinkedIn Sales Navigator |
| **Volumen** | 50-100 Kontakte/Woche |
| **Erfolgsrate** | 5-10% Antworten |

**Strategie:**
- Profil: "Förderberater Südtirol"
- Posts: 2-3x pro Woche
- Direktnachrichten: Personalisiert, kurz

---

### 3. Wirtschaftskammer

| | |
|---|---|
| **Methode** | Partnerschaft, Veranstaltungen |
| **Kontakt** | Wirtschaftskammer Südtirol (lvh) |
| **Nutzen** | Zugang zu Mitgliedern, Glaubwürdigkeit |

**Möglichkeiten:**
- Vorträge bei Info-Veranstaltungen
- Partner-Programm
- Empfehlungen

---

### 4. Lokale Presse

| | |
|---|---|
| **Methode** | Pressemitteilungen |
| **Kanäle** | Dolomiten, Südtiroler Tagblatt |
| **Nutzen** | Bekanntheit, Glaubwürdigkeit |

---

### 5. Mundpropaganda

| | |
|---|---|
| **Methode** | Zufriedene Kunden empfehlen |
| **Nutzen** | Beste Leads, günstigste Kundenakquise |

**Aktion:**
- Exzellenten Service bieten
- Um Bewerbung bitten (Google, LinkedIn)
- Referral-Programm (10% Rabatt auf nächstes Projekt)

## Vertriebs-Zeitplan

| Woch | Fokus | Aktivität |
|---|---|---|
| 1 | Vorbereitung | Listen erstellen, E-Mail-Vorlagen |
| 2 | E-Mail | 100 E-Mails raussenden |
| 3 | LinkedIn | Profil aufsetzen, 50 Kontakte |
| 4 | Follow-up | Antworten bearbeiten, Termine |
| 5 | E-Mail | Weitere 200 E-Mails |
| 6 | Scale | Mehr LinkedIn, Partnerschaften |
